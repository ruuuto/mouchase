こんにちは、ランディングページをご覧下さり誠にありがとうございます、
このコードは体験版のものとは大きく異なったものとなっています、
ですので、ファイルに書いてあるような記述を元にご自身（あるいはAI）でカスタマイズすることをおすすめします
このコードで使われているライブラリ・拡張機能は以下の通りです。
コア機能

MediaPipe (mediapipe) — Googleが開発した手のランドマーク検出ライブラリ。21点の手の骨格座標をリアルタイムで取得します
OpenCV (cv2) — カメラ映像の取得・加工・ウィンドウ表示に使用
PyAutoGUI (pyautogui) — マウス移動・クリック・スクロールなどのPC操作を自動化

補助的なもの

collections.deque — マウス座標のスムージング用に過去フレームの座標を保持するバッファとして使用
time — スクロールの連続実行を防ぐインターバル管理に使用
warnings / os — TensorFlow・ProtocolBuffers関連の不要な警告ログを抑制するために使用

特に中心となるのは MediaPipe + OpenCV + PyAutoGUI の3つで、「カメラで手を認識 → ジェスチャーを判定 → PCを操作する」という流れをそれぞれが担っています。
ですので、仕様にはこのようなライブラリ前提のものとなっています、pc操作の自動化においては、誤作動による暴走などにお気をつけください。
